    Plug QML Type

    Represents a smart plug device. More...

    • List of all members, including inherited members

    Detailed Description

    The Plug component provides a UI to control a smart plug. It shows whether the plug is turned on or off, and allows toggling its status.

    Properties

    The model object that contains this plug's information. For example:

    { "name": "Plug", "icon": IconType.plug, "settings": { "status": false } }
